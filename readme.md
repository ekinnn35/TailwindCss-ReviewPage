# TailwindCSS Review Page

Bu proje, **Tailwind CSS** kullanılarak oluşturulmuş modern ve responsive bir inceleme/yorum sayfasıdır. Projede temel olarak kullanıcı yorumlarının kart yapısında listelendiği, şık ve sade bir arayüz hedeflenmiştir.

## 🚀 Kullanılan Teknolojiler

- **HTML5**
- **Tailwind CSS** (CDN veya yerel yapılandırma)
- **Responsive Design**
- **Custom Google Fonts**
- **Minimal UI Design**

```

📦TailwindCss ReviewPage

## 🖼️ Sayfa Özellikleri

- Kullanıcı yorumları kart formatında görüntülenir.
- Her kartta profil görseli, isim, yorum ve yıldız derecelendirmesi bulunur.
- Mobil uyumludur ve farklı ekran boyutlarında optimize çalışır.
- `Tailwind` utility sınıfları kullanılarak tasarım sade tutulmuştur.
